 Minesweeper als Widget

Hier das von Windows bekannte Spiel "Minesweeper" als Widget.

Das aktuelle Spielfeld wird immer in der Datenbank gespeichert, somit kann ein Spiel jederzeit unterbrochen und später fortgesetzt werden.
An den Browser wird nur übermittelt was zur Anzeige notwendig ist. Cheaten sollte also ausgeschlossen sein.

Die Größe des Spielfeldes und die Anzahl der Bomben kann im oberen Bereich der Plugindatei verändert werden.

Bedienung: Wie bei Windows, per Klick / Rechtsklick.

Sprachen: Deutsch & Englisch

Aktuelle Version: 1.0.1 von 28.02.2009
