Einfaches Erstellen von Widgets mit beliebigem Inhalt über die Administration

Mit diesem Plugin kann man mit wenigen Klicks eigene Widgets erstellen um z.B. beliebige HTML-Codes einzubinden oder ein kleines Menü zu erstellen.

Features:
- Freie Verwendung von HTML- und Smarty(Template)-Code* als Inhalt
- Einstellbar ob das Widget für das Start- oder/und Organizer-Dashboard verwendbar ist
- Widgets werden als Plugin-Widgets "simuliert", somit ist auch die Nutzung von Widget-Layouts möglich
- Sprachen: Deutsch & Englisch (nur für Administration relevant)


*Hinweis: Da der eingegebene Code von Smarty interpretiert wird, kann es z.B. bei Verwendung von Javascript nötig sein den eingegebenen Code zwischen {literal} und {/literal} zu setzen im Fehler zu vermeiden.

Aktuelle Version: 1.0.1 von 21.03.2009

- Parsen durch Smarty kann deaktiviert werden für einzelne Widgets
- Es kann für die Widgets festgelegt werden für welche Gruppen sie verwendbar sind
